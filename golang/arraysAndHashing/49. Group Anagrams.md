# 49. Group Anagrams

## problem

將陣列的所有字串依照是否是同個 anagram 的分組, 並回傳分組後的結果

## thoughts

### sorting

將每個字串按照字母排序小到大, 用 map 記錄同個 anagram 的所有字串

map的結構: [str]: {str1, str2, str3, ...}

如果排序後的字串已經出現在 map 的 key 中, 則將原字串 append 到 value 的 slice 

最後將 map 的所有 slices 轉存到一個 slice of slice 回傳

### character count map

**如果兩個字串的字母出現次數一樣, 則代表兩個字串是 anagram**

遍歷每個字串, 將字母的出現次數記錄在一個 [26]int 的 array 中, 並將它當作 map 的 key, 用 map 記錄同個 anagram 的所有字串

因為 golang 的 array 是 value type, 陣列所有 index 的數值如果都一樣視為是相等的資料, 所以可以直接比對(也就是可以直接用 == 來比對)

map的結構: [[26]int]: {str1, str2, str3, ...}

如果該字串的字母出現次數陣列已經出現在 map 的 key 中, 則將原字串 append 到 value 的 slice

最後將 map 的所有 slices 轉存到一個 slice of slice 回傳
